# Places to visit in three of the capital cities of Pakistan.

I made this website to inform about  the places to visit in Pakistan.

##  About the Project

This repository contains a multi-page static website designed to educate users so that they have complete information whenever they visit Pakistan

*   **Lahore:** Places to  visit in Lahore.
*   **Karachi:** Places to visit in Karachi.
*   **Islamabad:** Places to visit in Islamabad.
